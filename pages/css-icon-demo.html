# 图标 [ Icon ] > 用一个标签元素代替一张图片
```css .picture.icon { color: #333; position: absolute; margin-left: 2px; margin-top: 2px; width: 105px; height: 105px; border-radius: 2px; overflow: hidden; border: solid 1px currentColor; box-shadow: 2px 2px 15px 10px #EEE; } .picture.icon:before { content: ''; ... } .picture.icon:after { content: ''; ... } ```
