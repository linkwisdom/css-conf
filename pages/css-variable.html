# 变量 [ variable ] > 通过变量，CSS获取更强的交互、适应和计算能力
- 定义变量 --xxx - 支持作用域 -tree-scope - 计算 calc: 支持 + - * / - JS可以改变样式变量
```css :root{ --font-color: #0C3934; --bg-color: #F8EBEE; } ```
- 定义css变量 ```js var root = document.documentElement; document.addEventListener('mousemove', evt => { let x = evt.clientX / innerWidth let y = evt.clientY / innerHeight root.style.setProperty('--mouse-x', x) root.style.setProperty('--mouse-y', y) }) ``` ```css .head:before { left: calc(20px + var(--mouse-x) * 20px); top: calc(20px + var(--mouse-y) * 20px); } .head:after { left: calc(60px + var(--mouse-x) * 20px); top: calc(20px + var(--mouse-y) * 20px); } ```
